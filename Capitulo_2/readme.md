# Anotaciones
    - HTML: HyperText Markup Languaje, sistema de marcas de texto
    - Estructura basica: HTML, HEAD Y BODY
    - Etiquetas comunes:
        - H1, H2 Y H3
        - DIV: Agrupa contenidos
        - P: Para colocar parrafos de texto
        - SPAN: Es un container generico, sin reglas preestablecidas

    - Atributos: Afectan al elemento por su presencia o enriquecen la defincion de esta.

    - CSS: Es la hoja de estilo en cascada, definen la apariencia del documento
        - Selector: Enlazar un estilo con un elemento del HTML
        - Atributos: Definen los estilos dentro de una clase

    - JavaScript: Permite realizar actividades complejas, controla las interacciones
        - DOM: Modelo objeto del documento
